# Predict-Customer-Churn-Web-App
A fully interactive web application built with Python, Streamlit, pandas, and scikit-learn that helps users analyze customer data and predict churn
The app allows users to:
1) Upload custom datasets in CSV format
2) Perform automated data cleaning, preprocessing, and optional feature engineering
3) Train and compare multiple machine learning models to identify the best-performing algorithm
4) Make predictions on new customer data
5) Generate churn probabilities for each customer with clear, interpretable outputs

This tool provides an end-to-end workflow—from raw data to actionable insights—making churn analysis simple, fast, and accessible.
